A gui client for [scoop](http://scoop.sh) - _a command-line installer for Windows_.

Installation
-
```
$ scoop bucket add scoop-viewer-bucket https://github.com/prezesp/scoop-viewer-bucket.git
$ scoop install scoop-viewer
```

Launch:
-
```
$ scoop gui
```

or 
```
$ scoopviewer
```
